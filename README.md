# RennerFotos
See photos of real people using ors product.

## rennerfotos-robot
### How run?
- `cd rennerfotos-robot`
- `npm install`
- `npm start`

### 3rd party
- [venom-bot](https://www.npmjs.com/package/venom-bot)

## rennerfotos-web (A.k.a Frontend)
### How run?
- `cd rennerfotos-web`
- `npx lite-server`
