# RennerFotos
See photos of real people using ors product.

## Requeriments
- [Node.JS](https://nodejs.org/en/) _tested v10.19.0_

## rennerfotos-robot
### Environment vars
Use .env
- MONGODB_URI
- POS_VENDA_NUMERO

### How run?
- `cd rennerfotos-robot`
- `npm install`
- `npm start`

### 3rd party
- [venom-bot](https://www.npmjs.com/package/venom-bot)

## rennerfotos-backend
### Environment vars
Use .env
- MONGODB_URI

### How run?
- `cd rennerfotos-backend`
- `npm install`
- `npm start`

## rennerfotos-web (A.k.a Frontend)
### How run?
- `cd rennerfotos-web`
- `npx lite-server`
